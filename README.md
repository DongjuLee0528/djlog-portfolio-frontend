# Portfolio Frontend

개인 포트폴리오 웹사이트(Frontend)입니다.  
프로젝트 소개, 기술 스택, 작업물(프로젝트), 연락처 정보를 제공합니다.


## Tech Stack
- Framework: (React / Next.js / Vue 중 선택)
- Language: (TypeScript / JavaScript)
- Styling: (TailwindCSS / CSS Modules / Styled-Components)
- State / Data: (React Query / Redux / Zustand 등)
- Build / Deploy: (Vercel / Netlify / GitHub Pages)

## Features
- 홈(소개)
- 프로젝트 목록 및 상세
- 기술 스택/경험 정리
- 연락처(이메일, GitHub, SNS)
- 반응형 UI

## Project Structure
```txt
src/
  assets/         # 이미지, 아이콘 등 정적 리소스
  components/     # 재사용 컴포넌트
  pages/          # 라우팅 페이지(또는 app/ if Next.js)
  styles/         # 스타일 관련
  api/            # API 호출 모듈(axios/fetch)
  hooks/          # 커스텀 훅
  utils/          # 유틸 함수
